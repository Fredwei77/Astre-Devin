# 易经常见问题功能说明

## 📋 功能概述

在易经占卜页面的问题输入框下方添加了8个常见问题标签，用户可以点击快速填入问题，也可以在填入后继续编辑。

## ✨ 功能特点

### 1. 快速选择
- 💡 8个精心设计的常见问题
- 🎯 涵盖事业、感情、财运等主要领域
- 👆 一键点击即可填入

### 2. 灵活编辑
- ✏️ 填入后可以继续编辑
- 🔄 可以多次点击不同问题
- 📝 支持完全自定义问题

### 3. 视觉反馈
- ✨ 悬停时高亮显示
- 🎨 点击时缩放动画
- 💫 平滑的过渡效果

### 4. 多语言支持
- 🌍 简体中文
- 🌍 繁体中文
- 🌍 英文


### JavaScript逻辑
```javascript
// 问题标签点击事件
document.querySelectorAll('.question-tag').forEach(tag => {
    tag.addEventListener('click', (e) => {
        e.preventDefault();
        const question = tag.dataset.question;
        const questionInput = document.getElementById('questionInput');
        if (questionInput && question) {
            questionInput.value = question;
            questionInput.focus();
            // 添加视觉反馈
            tag.style.transform = 'scale(0.95)';
            setTimeout(() => {
                tag.style.transform = '';
            }, 150);
        }
    });
});
```

## 💡 设计理念

### 为什么添加常见问题？

1. **降低使用门槛**
   - 新用户不知道如何提问
   - 提供参考模板
   - 引导正确使用

2. **提升效率**
   - 快速选择常见问题
   - 节省输入时间
   - 提高使用频率

3. **规范问题格式**
   - 示范清晰的提问方式
   - 避免模糊问题
   - 提升占卜质量

4. **增强用户体验**
   - 减少思考时间
   - 降低决策负担
   - 提供即时满足

### 问题选择标准

选择这8个问题的原因：

   - 事业（4个）

4. **适用性广**
