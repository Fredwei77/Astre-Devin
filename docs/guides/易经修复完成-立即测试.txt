技术说明
========================================

修改前:
const question = tag.dataset.question;
// 总是返回硬编码的中文

修改后:
const questionSpan = tag.querySelector('[data-i18n]');
const question = questionSpan.textContent.trim();
// 获取已翻译的显示文本
