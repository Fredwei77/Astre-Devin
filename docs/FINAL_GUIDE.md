## 🚀 立即开始使用

### 方法 1: 使用模拟模式（推荐新手）

**无需 API Key，立即体验！**

1. 打开浏览器访问：
   ```
   http://localhost:8000/
   ```

2. 点击右上角语言切换器，选择"简体中文"

3. 体验三大功能：
   - 🔮 占卜：http://localhost:8000/divination.html
   - 🧭 风水：http://localhost:8000/fengshui.html
   - ☯️ 易经：http://localhost:8000/iching.html

**注意：** 模拟模式使用预设数据，不消耗 API 额度

### 方法 2: 使用真实 API

**获得真实 AI 分析！**

1. 获取 OpenRouter API Key：
   - 访问 https://openrouter.ai/
   - 注册并登录
   - 创建 API Key

2. 配置 API Key：
   - 打开 `config.js`
   - 找到第 6 行
   - 替换为你的 API Key：
   ```javascript
   OPENROUTER_API_KEY: 'sk-or-v1-你的密钥',
   ```

3. 关闭模拟模式：
   - 找到第 33 行
   - 改为：
   ```javascript
   MOCK_MODE: false
   ```

4. 刷新页面，开始使用！

## 🔧 常见问题

### Q1: 点击按钮没反应？
**A:** 
1. 按 `Ctrl + Shift + R` 强制刷新
2. 检查浏览器控制台（F12）
3. 确认 `config.js` 和 `ai-service.js` 已加载

### Q2: 翻译没有生效？
**A:**
1. 刷新页面
2. 检查 `language-switcher-final.js` 是否加载
3. 查看浏览器控制台错误

### Q3: API 请求失败？
**A:**
1. 检查 API Key 是否正确
2. 确认账户有余额
3. 尝试使用模拟模式测试

### Q4: 响应很慢？
**A:**
1. 正常，AI 分析需要 5-15 秒
2. 可以切换到 Gemini Flash（更快）
3. 或使用模拟模式（即时）

### Q5: 如何切换模型？
**A:**
编辑 `config.js`：
```javascript
MODELS: {
    DIVINATION: 'google/gemini-flash-1.5',  // 更快
    FENGSHUI: 'google/gemini-flash-1.5',
    ICHING: 'google/gemini-flash-1.5'
}
```
