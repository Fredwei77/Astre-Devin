# 易经卦前提示优化报告

## 📋 优化需求

将易经页面的"选择您的占卜方法"部分替换为"卦前提示"，添加七条传统易经占卜的注意事项，帮助用户以正确的心态和状态进行占卜。

## 🎯 优化目标

1. **传承传统文化**：展示易经占卜的传统规矩和礼仪
2. **引导正确心态**：帮助用户以虔诚、专注的态度进行占卜
3. **提升占卜质量**：确保用户在最佳状态下进行占卜
4. **增强仪式感**：强调易经占卜的严肃性和神圣性

### 2. 添加七条占卜规则

#### 1️⃣ 无事不占
**含义**：卜卦是为了在迷茫时更好地抉择，成为更好的自己，顺其自然为上，先尽人事，后听天命。

**解释**：
- 不要为琐碎小事占卜
- 占卜应用于重要决策
- 先努力尝试，再寻求指引
- 保持顺其自然的心态

#### 2️⃣ 体耗不占
**含义**：饮酒、性生活、剧烈运动后，在状态未恢复前尽量不进行易占。

**解释**：
- 身体状态影响占卜准确性
- 需要充沛的精力和体力
- 避免在体力透支时占卜
- 等待身体恢复到最佳状态

#### 3️⃣ 心疲不占
**含义**：过分疲劳或精神、心理状态不佳时尽量不进行易占。

**解释**：
- 精神状态影响感应能力
- 疲劳会降低专注力
- 心理压力影响判断
- 需要清醒的头脑

#### 4️⃣ 天异不占
**含义**：气候异常、环境特别和节令交替等特别时间里，尽量不进行易占。

**解释**：
- 天地能量在特殊时期不稳定
- 节气交替时气场变化
- 异常天气影响感应
- 选择平和的时间占卜

#### 5️⃣ 疑则不占
**含义**：心诚则灵，如果内心还持有较大的怀疑，不建议进行易占。

**解释**：
- 虔诚的心态最重要
- 怀疑会影响结果
- 需要相信易经智慧
- 半信半疑不如不占

#### 6️⃣ 避众而占
**含义**：尽量不在三人以上的场合为他人易占。

**解释**：
- 保持占卜的私密性
- 避免他人干扰
- 尊重占卜的严肃性
- 一对一更加专注

#### 7️⃣ 静心而占
**含义**：找一个安静的环境，不在易占的同时做与易占无关的事情。

**解释**：
- 环境安静有助于专注
- 排除外界干扰
- 全神贯注于占卜
- 一心一意求问

## 💡 设计理念

### 为什么添加卦前提示？

1. **尊重传统**
   - 易经占卜有严格的规矩
   - 传承千年的智慧
   - 不是随意的游戏
   - 需要虔诚的态度

2. **提升质量**
   - 正确的心态影响结果
   - 合适的时机很重要
   - 专注的环境必不可少
   - 身心状态决定感应

3. **教育用户**
   - 了解易经文化
   - 学习占卜礼仪
   - 建立正确认知
   - 避免滥用占卜

4. **增强体验**
   - 仪式感提升参与度
   - 严肃性增加可信度
   - 文化感丰富内涵
   - 神圣感强化效果

### 七条规则的意义

这七条规则来自传统易经占卜的实践经验，每一条都有深刻的道理：

1. **无事不占** - 避免滥用，保持敬畏
2. **体耗不占** - 身体状态影响感应
3. **心疲不占** - 精神状态决定质量
4. **天异不占** - 天地能量影响结果
5. **疑则不占** - 心诚则灵的体现
6. **避众而占** - 保持专注和私密
7. **静心而占** - 环境和心态的要求
