# 占卜深挖真相AI对话框提示词优化完成

## 优化内容

### 1. 新增深度分析系统提示词 (config.js)

在 `CONFIG.PROMPTS.DIVINATION` 中新增 `FOLLOWUP_SYSTEM` 函数，支持中英文双语：

**核心特点**：
- 基于 Prompts.txt 文档的专业命理学框架
- 引用经典命理典籍：《滴天髓》、《千里命稿》、《子平真诠》、《三命通会》等
- 结合现代心理学（荣格原型、MBTI）和面相学
- 提供古文与白话文结合的专业分析

**分析框架**（5大部分）：
1. **命局总评与原局分析**
   - 八字排盘（年月日时四柱、胎元、命宫、身宫）
   - 强弱喜忌分析（日主得令得地得势、格局判定、喜用神忌神）
   - 四宫六亲分析（父母宫、事业宫、婚姻宫、子女宫）
   - 神煞分析（天乙贵人、桃花、羊刃、空亡等）

2. **流年运势（2026丙午年）**
   - 流年总纲（大运与流年的刑冲合害关系）
   - 十二流月详解（农历正月至十二月的事业财运健康感情）
   - 引用经典断语佐证

3. **生存指南与改运操作**
   - 风水调节（颜色、方位、摆件）
   - 生活细节（饮食、穿搭、饰品材质）
   - 心理与行为建议（结合MBTI分析性格盲点）

4. **社交与婚恋画像**
   - 贵人与小人（具体八字特征，如"喜见天干透丙火之人"）
   - 婚配画像（外貌、年龄差、职业、性格、相处模式）

5. **约束条件**
   - 术语通俗化解释
   - Markdown格式，重点加粗
   - 客观中立，提供化解方案

### 2. 优化追问上下文构建 (divination-followup.js)

**buildFollowupContext 函数增强**：
- 提取并传递五行分布数据
- 包含完整的出生信息（日期、时间、地点、性别、四柱）
- 传递之前的占卜摘要作为上下文
- 明确用户追问的问题
- 要求按照5大分析框架输出详细报告

**中文提示词示例**：
```
请根据以下出生信息回答用户的追问，并进行深度八字分析。

**Task（任务）**：根据下方的出生信息，进行深度的八字排盘与运势推演...

**Input Data（命主信息）**：
- 性别：男/女
- 出生地点：北京（用于校对真太阳时）
- 出生日期：1990-01-01
- 出生时间：08:30
- 四柱：年【庚午】月【戊子】日【甲寅】时【戊辰】

**分析类别**：事业

**之前的占卜摘要**：...

**五行分布**：木: 70, 火: 45, 土: 80, 金: 60, 水: 35

**用户的追问**：如何提升我的事业运势？

**Analysis Framework（分析框架&输出要求）**：
[详细的5大部分分析要求]
```

### 3. 升级追问处理逻辑 (divination-followup.js)

**handleFollowupQuestion 函数改进**：
- 获取优化的系统提示词 `CONFIG.PROMPTS.DIVINATION.FOLLOWUP_SYSTEM(lang)`
- 使用新增的 `AIService.chatWithSystem()` 方法
- 分离系统提示词和用户提示词，提供更精准的AI指令
- 保持多语言支持（中文/英文）

### 4. 新增AI服务方法 (ai-service.js)

**新增两个方法**：

1. **chatWithSystem(systemPrompt, userPrompt, options)**
   - 支持自定义系统提示词的通用聊天接口
   - 用于深挖真相等需要专业系统提示词的场景
   - 返回AI响应文本

2. **chat(message, options)**
   - 简化的聊天接口
   - 使用默认系统提示词
   - 适用于简单追问场景
