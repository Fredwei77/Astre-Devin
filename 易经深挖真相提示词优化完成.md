# 易经深挖真相AI对话框提示词优化完成

## 优化内容

### 1. 新增深度分析系统提示词 (config.js)

在 `CONFIG.PROMPTS.ICHING` 中新增 `FOLLOWUP_SYSTEM` 函数，支持中英文双语：

**核心特点**：
- 基于 Prompts.txt 文档的专业易经学框架
- 引用经典易经典籍：《周易本义》、《易传》（十翼）、《程氏易传》、《朱熹易学》、《王弼易注》等
- 结合儒释道三家哲学、荣格心理学（同步性理论）和现代决策科学
- 提供古老易经智慧与现代人生指导结合的实用建议

**分析框架**（6大部分）：
1. **卦象结构与象征分析**
   - 分析上下卦、五行属性、相生相克关系
   - 阐释卦名由来和象征意义

2. **卦辞与象辞解读**
   - 引用并解释原文卦辞和象辞
   - 用白话文阐释含义，结合用户问题

3. **变爻分析与转化**
   - 分析每个变爻的含义
   - 解释本卦到之卦的转化
   - 揭示事态发展趋势

4. **形势判断与时机分析**
   - 判断当前形势（吉凶、时机、势态）
   - 提供时机分析（何时行动、何时等待、何时退守）

5. **行动指导与决策建议**
   - 结合易经智慧和现代决策科学
   - 提供具体可行的建议（应该做什么、避免什么、如何应对）

6. **心理洞察与心态调整**
   - 结合荣格心理学分析用户心理状态
   - 提供心态调整建议，帮助用户以平和心态面对局势

### 2. 优化追问处理逻辑 (iching.html)

**handleFollowupQuestion 函数增强**：
- 获取当前语言设置
- 提取完整的卦象数据：
  * 卦象编号
  * 卦象名称
  * 变爻位置
- 使用 `CONFIG.PROMPTS.ICHING.FOLLOWUP_SYSTEM(lang)` 获取优化系统提示词
- 调用新增的 `buildFollowupUserPrompt()` 方法构建用户提示词
- 使用 `formatFollowupAnswer()` 方法格式化AI回答

**buildFollowupUserPrompt 方法**：
- 提供完整的占卜上下文：
  * 原始问题
  * 卦象编号和名称
  * 变爻信息
  * 用户追问
- 要求按照6大分析框架输出详细报告
- 支持中英文双语

**formatFollowupAnswer 方法**：
- 将换行符转换为HTML
- 高亮关键词（卦象、卦辞、象辞、变爻、吉凶、时机等）
- 格式化列表
- 支持中英文关键词
